v2 → v3 で変えたこと:
  • グラデーション背景 → フラットなダーク (#0c0c0c) — 紫がかった gradient を排除
  • グラデーションテキスト → 単色テキスト (#f0f0f0) — 読みやすさ優先
  • 光彩ボケ (glow) → 削除 — 装飾的なだけのエフェクトを除去
  • ヒーローを左揃え — 中央揃え一辺倒をやめ、プロのLP感を出す
  • ブランドカラーの彩度を少し落とす — #01E246→#2dd864、#00A8FB→#3eb8f0
  • CTAボタン — グラデーションボタン→白背景ベタ塗り。注目を集めつつ落ち着く
  • カードの背景 — 透明度指定 → #161616 のソリッドカラー
  • セクション間はシンプルな 1px ディバイダー — 余白だけで区切る
  • カテゴリラベル — badge風背景色 → テキストカラーのみで分類
